CSS Selectors

An Element Selector is used to apply styles to all instances of a specific HTML tag.

An ID Selector targets a single, unique element on webpage, identified by a specific id attribute.

A Class Selector is used to apply styles to multiple elements that share same class attribute.

This heading and the heading below are styled using grouping selector.

The Grouping Selector allows to apply the same styles to multiple elements.

The Universal Selector is used to select and apply styles to all elements on a webpage.
